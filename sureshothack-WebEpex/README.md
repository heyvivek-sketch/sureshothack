## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- **Landing Page** (`/`) - Main dashboard with game selection
- **Login Page** (`/login`) - User authentication
- **Sign Up Page** (`/signup`) - New user registration

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom components with Tailwind

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Design

The design features:
- Dark gradient backgrounds with orange and cyan glows
- Orange accent color (#FF6B35) for primary actions
- Dark gray cards with subtle borders
- Smooth transitions and hover effects
- Responsive layout for all screen sizes
